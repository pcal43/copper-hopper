Copper Hopper adds a new kind of hopper that you can use to filter which items get pulled and pushed. They help you more easily build item sorting systems while keeping the spirit of vanilla Minecraft.

Features

  • Filter up to 5 items
  • Filter both stackable and unstackable items
  • Works with the Fabric modloader
  • Can work without being installed in the client (thanks to the awesome Polymer framework)
  • Made of copper!
